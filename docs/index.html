Introduction

These pages describe the RADx Metadata Specification. This metadata specification contains a meta template that specifies required, recommended and optional fields that are used for capturing metadata that described data files submitted to the RADx Data Hub.

We recommend that you start by reading the specification. After this, you can learn about the different ways of creating metadata instances that describe individual data files and comply with the specification that is described here.
